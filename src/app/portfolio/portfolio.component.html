EvalOn – Teacher Evaluation System

Full-stack evaluation platform using Angular, Tailwind CSS, PHP, Python (NLP) & MySQL.

EvalOn is a school evaluation system designed to help students evaluate teachers through a structured form. Java & JavaFX desktop app with QR (ZXing) and MySQL integration.

This system is a desktop application that helps landlords manage tenants, payments, rooms, and rental history. QR codes are generated using ZXing to make tenant identification faster and more secure. MySQL stores all rental records, ensuring efficient data tracking and reporting.

MayWeather – Weather Mobile App

Built using Ionic, SQLite & a weather API with offline caching and clean UI.

MayWeather is a mobile app that provides real-time weather data and forecasts. It stores recent weather results using SQLite, allowing the app to work even without an internet connection. Designed with a clean UI, it focuses on accessibility and fast information delivery for mobile users.
